# 🧪 AI 바이브 코딩 실험

## 대마코인 x ChatGPT 게임 개발 실험

---

### 📋 실험 개요

> **하루에 ChatGPT로 만든 프로젝트 10개를 GitHub에 올리는 실험**

이 저장소는 **AI 바이브 코딩 실험**을 위한 프로젝트입니다.
매일 ChatGPT를 활용하여 10개의 완성된 게임을 생성하고, GitHub에 커밋하는 과정을 기록합니다.

---

### 🎯 실험 목적

1. **AI 생성 코드의 실제 동작 여부 확인** — ChatGPT가 만든 코드가 실제로 작동하는지 검증
2. **생산성 측정** — 하루에 AI로 몇 개의 프로젝트를 만들 수 있는지 테스트
3. **코드 품질 분석** — AI가 생성한 코드의 구조, 가독성, 완성도 평가
4. **개발 워크플로우 탐색** — AI와 협업하는 새로운 개발 방식 연구

---

### 📅 실험 방식

```
시간: 약 10분 (10개 게임 생성 + 커밋)
도구: ChatGPT (Selenium 자동화 크롤러)
방법: 멀티 에이전트 10개 동시 실행
결과: 각 게임당 1개 커밋 → 하루에 10개 커밋
```

| 단계 | 내용 | 소요 시간 |
|------|------|-----------|
| 1단계 | ChatGPT에 게임 제작 프롬프트 전송 | ~5초 |
| 2단계 | ChatGPT가 HTML/CSS/JS 코드 생성 | ~30초 |
| 3단계 | 생성된 코드 추출 및 저장 | ~5초 |
| 4단계 | GitHub에 커밋 및 푸시 | ~2초 |
| **총 소요** | **10개 게임 동시 생성** | **약 6~7분** |

### 🛠️ 사용 기술

- **언어**: HTML5, CSS3, JavaScript (Canvas API)

### 🚀 실행 방법

각 게임은 단일 HTML 파일로 구성되어 있어 브라우저에서 바로 실행 가능합니다.

```bash

open 11_tetris/index.html           # macOS
start 11_tetris/index.html          # Windows
```
